# Laravel Coolify Documentation

Documentation site built with [Astro Starlight](https://starlight.astro.build/).

## Development

```bash
cd docs
npm install
npm run dev
```

Opens at http://localhost:4321/laravel-coolify

## Build

```bash
npm run build
```

Output in `docs/dist/`.

## Deployment

Automatically deployed to GitHub Pages on push to `main` via `.github/workflows/deploy-docs.yml`.

Live at: https://stumason.github.io/laravel-coolify
